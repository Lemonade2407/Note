# 数据类型
### 数字类型
#### 整数类型

| 进制   | 引导符号(不区分大小写) |
| ---- | ------------ |
| 十进制  | 无            |
| 二进制  | 0b           |
| 八进制  | 0o           |
| 十六进制 | 0x           |
#### 浮点类型
- 高精度浮点运算：decimal库
- 定义：`a = decimal.Decimal('高精小数')`
- 自定义位数：`decimal.getcontext().prec = 位数`
#### 复数类型
- 虚部通过后缀“J”或“j”表示
- 对于复数z，`实部 = z.real 虚部 ＝ z.imag`

### 字符串类型及其操作

#### 符号体系
- 正向递增序号：字符串从左到右，序号从0递增
- 反向递减序号：字符串从右到左，序号从-1递减
#### 操作符
- 切片：`str[N:M]`,N,M可缺失
#### 操作函数

| 函数     | 描述   |
| ------ | ---- |
| len(x) | 返回长度 |
#### 处理方法

| 方法                         | 描述                                |
| -------------------------- | --------------------------------- |
| str.split(sep,maxsplit)    | 返回一个列表，包含str根据sep分割的部分            |
| str.strip(chars)           | 返回str，两侧去掉chars中列出的字符             |
| str.center(width,fillchar) | 返回长度为width的字符串，str居中，两侧填充fillchar |
| str.zfill(width)           | 返回长度为width的字符串，str不足部分左侧填0        |
| str.format()               | str中用占位符{顺序编号}占位，方法中参数按顺序填充       |

# 程序结构
## 分支结构
```
if 条件1:
	语句块1
elif 条件2:
	语句块2
else:
	语句块n
```
## 循环结构
### 遍历循环
```
for 变量 in 遍历结构:
	语句块
else:
	正常结束后语句
```
常见用法
```
遍历N次，从1～(N-1)
for i in range(N):
	语句块
```
### 无限循环
```
while 条件:
	语句块
else:
	正常结束后语句
```
## 异常处理
```
try：
	语句块
except 异常类型1:
	语句块
except:
	语句块	
finally：
	无论是否报错都执行的语句块
```

# 组合数据类型
## 序列类型
- 元素有序
- 字符串(str)、元组(tuple)、列表(list)
- 元组一旦创建不能修改，用()表示
- 列表与数组相比，内存动态分配，数据类型不必一致
## 集合类型
- 元素无序且唯一
- 集合(set)，用{}表示
## 映射类型
- 键值对
- 字典(map)
- `创建：字典名 ＝ {k1:v1,k2:v2...}`
- `查询：值 ＝ 字典名[键值]`
