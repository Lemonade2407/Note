# 语法
## 数据类型
### 数字类型
#### 整数类型

| 进制   | 引导符号(不区分大小写) |
| ---- | ------------ |
| 十进制  | 无            |
| 二进制  | 0b           |
| 八进制  | 0o           |
| 十六进制 | 0x           |
#### 浮点类型
- 高精度浮点运算：decimal库
- 定义：`a = decimal.Decimal('高精小数')`
- 自定义位数：`decimal.getcontext().prec = 位数`
#### 复数类型
- 虚部通过后缀“J”或“j”表示
- 对于复数z，`实部 = z.real 虚部 ＝ z.imag`
### 数字类型的操作
#### 运算操作符

#### 运算函数

| 函数        | 描述          |
| --------- | ----------- |
| abs(x)    | x的绝对值       |
| pow(x,y)  | x**y        |
| roud(x,n) | x四舍五入保留n位小数 |
