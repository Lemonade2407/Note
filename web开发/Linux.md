# 常用命令
- 格式：command 【-options选项】【parameter参数】
- 技巧：tab自动补全，箭头调出曾用命令，clear或Ctrl+l快速清屏
- command --help可输出指令帮助
## 目录操作命令
- ls：显示指定目录下的内容
	- ls -al dir
	- -a：显示所有文件及目录，包括隐藏文件
	- -l：列出文件名、类型、权限、拥有者、大小等详细信息，可简写为ll
	- dir：指定目录，默认当前目录
- cd：切换路径
	- cd dirName
	- tips：
		- .表示目前所在目录
		- ..表示上一级目录
		- ~表示home目录
		- -表示上一次所在目录
	- dirName：绝对路径/相对路径
- mkdir：创建目录
	- mkdir -p dirName
	- -p：确保目录名存在，若不存在则创建一个
- rm：删除文件或目录
	- rm -rf name
	- -r：将目录及其中内容全部删除
	- -f：强制删除，无需确认
## 文件操作命令
- cat：显示文件所有内容，一般用于看小文件
	- cat -n fileName
	- -n：对行数编号（从1开始） 
- more：以分页的形式展示文件内容
	- more fileName
	- tips：
		- 回车：向下滚动一行
		- 空格：向下滚动一屏
		- b：返回上一屏
		- q/Ctrl+c：退出
- head：查看文件开头内容
	- head -n fileName
	- -n：输出开头n行内容
- tail：查看文件尾部内容
	- tail -nf fileName
	- -n：输出末尾n行内容
	- -f：动态读取末尾内容并展示，一般用于日志输出
## 拷贝移动命令
- cp：复制文件或目录
	- cp -r source dest
	- -r：复制该目录下所有内容
- mv：为文件或目录重命名、或移动到其他位置
	- mv source dest
## 打包压缩命令
- tar：对文件打包、解包、压缩、解压
	- tar -zcxvf fileName files
	- -z：压缩或解压
	- -c：创建新包（打包）
	- -x：从包中还原文件（解包）
	- -v：显示命令执行过程
	- -f：用于指定包文件的名称
	- tips：
		- .tar表示只打包不压缩
		- .tar.gz表示既打包也压缩
## 文本编辑命令
- vi/vim：类似记事本
	- vi/vim fileName
	- 安装vim：yum install vim
	- 命令模式：
		- gg：定位第一行
		- G：定位最后一行
		- dd：删除所在行
		- ndd：删除所在行及之后n行数据
		- u：撤销
		- 命令模式->插入模式：i,a,o
		- 命令模式->底行模式：:
	- 底行模式：
		- :wq：保存并退出
		- :q!：不保存并退出
		- :set nu：显示行号
		- :set nonu：取消行号显示
		- :n：定位到第n行
