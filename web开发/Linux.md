# 常用命令
- 格式：command 【-options选项】【parameter参数】
- 技巧：tab自动补全，箭头调出曾用命令，clear或Ctrl+l快速清屏
- command --help可输出指令帮助
- ls：显示指定目录下的内容
	- ls -al dir
	- -a：显示所有文件及目录，包括隐藏文件
	- -l：列出文件名、类型、权限、拥有者、大小等详细信息，可简写为ll
	- dir：指定目录，默认当前目录
- cd：切换路径
	- cd dirName
	- tips：
		- .表示目前所在目录
		- ..表示上一级目录
		- ~表示home目录
		- -表示上一次所在目录
	- dirName：绝对路径/相对路径
- mkdir：创建目录
	- mkdir -p dirName
	- -p：确保目录名存在，若不存在则创建一个
- rm：删除文件或目录
	- rm -rf name
	- -r：将目录及其中内容全部删除
	- -f：强制删除，无需确认
