# 会话技术
- 用户访问服务器，直到一方断开连接为一次会话，一次会话可以有多次请求
- 会话跟踪：服务器识别多次请求是否来自同一用户
	- 客户端会话跟踪技术：Cookie
	- 服务端会话跟踪技术：Session
	- 令牌技术
## Cookie
- 优点：HTTP协议支持
- 缺点：
	- 移动端无法使用
	- 不安全，用户可修改
	- 不能跨域（协议、IP、端口任一不同）
- 设置Cookie(向前端响应)
```
@GetMapping("")
public Result cookie1(HttpServletResponse response){
	response.addCookie(new Cookie("name","value"));
	return Result.success();
}
```
- 获取Cookie(从前端获取请求)
```
@GetMapping("")
public Result cookie2(HttpServletRequest request){
	Cookie[] cookies = request.getCookies();
	for(Cookie cookie : cookies){
		if(cookie.getName().equals("name"))
		System.out.println("name:"+cookie.getValue());
	}
	return Result.success();
}
```
## Session
- 底层基于Cookie，通过会话ID标识会话而不是name
- 优点：存储在服务器，安全
- 缺点：
	- 集群服务器无法使用
	- Cookie的缺点
- 设置Session(向前端响应)
```
@GetMapping("")
public Result session1(HttpSession session){
	log.info("HttpSession-s1:{}",session.hashCode);
	session.setAttribute("loginUser","userName")
	return Result.success();
}
```
- 获取Session(从前端获取请求)
```
@GetMapping("")
public Result session2(HttpSession session){
	log.info("HttpSession-s1:{}",session.hashCode);
	Object loginUser = session.getAttribute("loginUser","userName");
	log.info("loginUser:{}",loginUser);
	return Result.success(loginUser);
}
```
## 令牌
- 优点：
	- 支持移动端和PC端
	- 解决集群环境的认证问题
	- 减轻存储压力
- 缺点：自己实现
### JWT令牌
- JSON Web Token
- 组成：
	- Header（头）：记录令牌类型、签名算法等
	- Payload（有效载荷）：携带信息
	- Signature（签名）：融入前两部分和密钥，经算法得来
- 生成/解析：引入jjwt依赖
### 令牌校验
#### 过滤器Filter
- 拦截对资源的请求
```
@WebFilter(urlPatterns = "/*")//拦截所有请求
public class TokenFilter implements Filter{
	//初始化
	public void init(FilterConfig filterConfig) throws ServletException {
		System.out.println("init");
	}
	//核心功能
	public void doFilter(ServletRequest servletRequest,ServletResponse servletResponse,FilterChain filterChain) throws ServletException {
		HttpServlentRequest request = (HttpServletRequest) servletRequest;
		HttpServlentRsponse response = (HttpServletResponse) servletResponse;
		//获取请求路径
		String requestURI = request.getRequestURI();
		//判断是否是登录请求
		if(requestURI.contains("/login")){
			filterChain.doFilter(request,response);
			return;
		}
		//获取token
		String token = request.getHeader("token");
		//判断token是否存在
		if(token == null||token.isEmpty()){
			response.setStatus(401);
			return;
		}
		try{
			JwtUilts.parseToken(token);
		}catch(Exception e){
			response.setStatus(401);
			return;
		}
	}
	//销毁
	public void destroy(){
		System.out.println("destroy");
	}
}
```
