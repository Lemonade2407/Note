# 绪论
## 数据库系统概述
### 4个基本概念
1. 数据Data
2. 数据库DB
3. 数据库管理系统（数据管理软件）
4. 数据库系统（数据库+数据库管理系统+应用程序+数据库管理员）DBS
### 数据管理技术的产生和发展
1. 人工管理阶段
	1. 数据不保存
	2. 应用程序管理数据
	3. 数据不共享
	4. 数据不具有独立性
2. 文件系统阶段
	1. 数据长期保存
	2. 数据由专门软件管理
	3. 数据共享性弱，冗余度高
	4. 数据独立性弱
3. 数据库系统阶段
	1. 整体数据结构化：主要特征与本质区别
	2. 数据的共享性强、冗余度低且易于扩充
	3. 数据的独立性强：显著优点
		1. 物理独立性
		2. 逻辑独立性
	4. 数据由数据库管理系统统一管理和控制
