- controller
```
//获取文件的形参为MultipartFile类
public Result handleFileUpload(MultipartFile file)
{
	//本地存储
	//获取原始文件名
	String originalFilename = file.getOriginalFilename();
	//生成唯一文件名
	String extension = originalFilename.substring(originalFilename.lastIndexOf("."));//获取后缀
	String newFileName = UUID.randmUUID().toString()+extension;//随机UUID文件名
	//保存文件
	file.transferTo(new File("path"))
}
```
- 配置（application文件内）
```
servlet:
 multipart:
  max-file-size: 10MB//最大单个文件大小
  max-request-size: 100MB//最大请求大小
```
